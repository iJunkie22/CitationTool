Citation Tool

CitationTool is a C# program that provides a convenient way to cite your references. It is developed and debugged using Visual Studio 2015 Community edition.

Current Features

  • Supports MLA and APA
  • Supports Book and Ebook

Features To Come

  • Film, Journal, Magazine, Website, Wiki citations
  • Support Chicago Style
